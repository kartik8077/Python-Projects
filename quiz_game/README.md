# 🖥️ The Ultimate Tech Quiz

A sleek, color-coded terminal-based Quiz Game built with Python. Test your knowledge on Computer Science topics ranging from PGP encryption to hardware history!

## ✨ Features
* **Dynamic UI:** Uses ANSI color codes for a professional terminal experience.
* **HTML Unescaping:** Automatically cleans up OpenDB API data (converts `&quot;` to real quotes).
* **Interactive Feedback:** Instant "Correct/Wrong" status with color-coded score tracking.
* **ASCII Splash Screen:** A retro-cool welcome screen upon launch.

## 📁 Project Structure
The project is organized within the `quiz_game/` directory:
* `main.py`: The entry point that runs the game loop.
* `quiz_brain.py`: Handles the game logic, scoring, and UI formatting.
* `question_model.py`: Defines the `Question` class structure.
* `data.py`: Contains the dictionary of quiz questions and answers.

## 🚀 How to Run


    ```bash
    cd Python-Projects/quiz_game
    ```
3.  **Run the game:**


## 🛠️ Requirements
* **Python 3.x**
* **Terminal Support:** Works best in VS Code Terminal, Windows Terminal, or macOS/Linux Terminals that support ANSI colors.
